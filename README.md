# 🛍️ Customer Behaviour Analysis

A end-to-end **Data Analytics** project that explores retail customer shopping behaviour using **Python, SQL, and Power BI**. The project takes raw transactional data through cleaning and exploratory analysis in Python, answers key business questions with SQL, and visualizes insights in an interactive Power BI dashboard.

## 📌 Overview

Understanding *why* and *how* customers shop is critical for any retail business. This project analyzes a retail customer dataset to uncover patterns in spending, discounts, subscriptions, product preferences, and customer loyalty — helping stakeholders make data-driven decisions around marketing, pricing, and inventory.

## 🎯 Business Problem

The full business context and objectives for this analysis are documented in [`Business Problem Document.pdf`](./Business%20Problem%20%20Document.pdf).

## 📊 Dataset

**`Retail_Customer_Insights.csv`** contains customer-level transaction records with fields such as:

- `customer_id`, `gender`, `age_group`
- `item_purchased`, `category`
- `purchase_amount`, `discount_applied`
- `review_rating`, `shipping_type`
- `subscription_status`, `previous_purchases`

## 🔍 Key Business Questions Answered (SQL)

The SQL script (`Customer_Behaviour_Analysis.sql`) answers questions including:

1. What is the total revenue generated by male vs. female customers?
2. Which customers used a discount but still spent more than the average purchase amount?
3. Which are the top 5 products with the highest average review rating?
4. How does average purchase amount compare between standard and express shipping?
5. Do subscribed customers spend more than non-subscribers?
6. Which 5 products have the highest percentage of discounted purchases?
7. How do customers segment into New, Returning, and Loyal buyers?
8. What are the top 3 most purchased products within each category?
9. Are repeat buyers (5+ previous purchases) more likely to subscribe?
10. What is the revenue contribution of each age group?

## 🐍 Python Analysis

`Customer_Shopping_Behaviour_Analysis.ipynb` covers:

- Data cleaning (handling missing values, duplicates, and inconsistent entries)
- Exploratory Data Analysis (EDA) on customer demographics and purchase behaviour
- Visualizations highlighting trends in spending, discounts, and product categories

## 📈 Power BI Dashboard

`Customer_Behaviour_Dashboard.pbix` presents an interactive dashboard summarizing:

- Revenue trends by gender, age group, and category
- Discount and subscription impact on sales
- Top-performing products and customer segments

> Open the `.pbix` file in [Power BI Desktop](https://powerbi.microsoft.com/desktop/) to explore the dashboard interactively.

## 🚀 Getting Started

### Prerequisites

- [Python 3.x](https://www.python.org/) with Jupyter Notebook
- `pandas`, `numpy`, `matplotlib`, `seaborn`
- A SQL environment (e.g., PostgreSQL, MySQL) to run the `.sql` queries
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (Windows) to open the `.pbix` dashboard

### Installation

```bash
# Clone the repository
git clone https://github.com/PRASANNA-163/Customer_Behaviour_Analysis.git
cd Customer_Behaviour_Analysis

# Install Python dependencies
pip install pandas numpy matplotlib seaborn jupyter
```

### Usage

1. **Python EDA:** Open `Customer_Shopping_Behaviour_Analysis.ipynb` in Jupyter Notebook to explore the data cleaning and analysis steps.
2. **SQL Queries:** Load `Retail_Customer_Insights.csv` into your SQL database as a `customer` table, then run the queries in `Customer_Behaviour_Analysis.sql`.
3. **Power BI Dashboard:** Open `Customer_Behaviour_Dashboard.pbix` in Power BI Desktop to interact with the visual dashboard.

## 💡 Key Insights

- Spending patterns vary meaningfully across gender and age groups.
- Discount usage does not always correlate with lower spend — some discount users still spend above average.
- Subscription status and purchase frequency are linked to higher customer loyalty.
- A small set of products drive the majority of purchases within each category.
